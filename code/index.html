// HOUSE RULES · NON-NEGOTIABLE

CHAT
CODE.

Short and blunt. This is a chat-driven live show, not a debate club. Volunteer mods run the room. Be cool — we never have to say a word.

ENFORCED NIGHTLY · ONE WARNING · THEN PERMANENT BAN
// 01 — WELCOME

✓ Welcome.

  • Real questions, real reactions, real personality. Talk to us like a friend would.
  • Multiple languages. English is the chat default, but say hi in any language.
  • Banter, jokes, Pattaya humor. The villa is a room, not a corporate set.
  • Thanking the mods. They work for free — a thank-you costs nothing.
  • New viewers asking obvious questions. Everyone was new once.
// 02 — BANNED IMMEDIATELY

✗ Banned.

  • Spam, self-promo links, "follow me" begging. Instant ban.
  • Racism, slurs, harassment, bigotry. Any flavor, any target.
  • Political fights or religious arguments. Not the room for it.
  • Explicit / sexual content in chat. We're a public broadcast.
  • Arguing with the mods. They're volunteers. Respect them or leave.
  • Asking for handouts. We're not a charity, this is a show.

One warning. Then ban.

We don't negotiate, we don't unban based on apologies, and we don't read the DMs.
