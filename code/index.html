HOUSE RULES.

Short and blunt. This is a chat-driven live show, not a debate club.

✓ WELCOME

  • Real questions, real reactions, real personality.
  • Multiple languages — English is the chat default, but say hi in any language.
  • Banter. Jokes. Local Pattaya humor.
  • Tipping the mods with a thank-you. They work for free.

✗ BANNED IMMEDIATELY

  • Spam, self-promo links, "follow me" begging.
  • Racism, slurs, harassment, bigotry. Any flavor.
  • Political fights or religious arguments.
  • Explicit / sexual content in chat.
  • Arguing with the mods. They're volunteers — respect them.

One warning. Then ban. We don't negotiate.
